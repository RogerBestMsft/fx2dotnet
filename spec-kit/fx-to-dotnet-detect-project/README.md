# fx-to-dotnet-detect-project — Project Type Detector

Read a project file and determine whether it is a web application host, web library, Windows Service, or another project type, and whether it uses SDK-style format.

## Command

`speckit.fx-to-dotnet-detect-project.detect` — Specify the `.csproj`, `.vbproj`, or `.fsproj` path to classify.

## Prerequisites

None — this extension is self-contained.

## State Files

None — returns classification results to the caller.
